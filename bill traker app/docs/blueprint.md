# **App Name**: Offline Bill Boss

## Core Features:

- Cinematic Intro: Implement a cinematic intro sequence on app startup with a logo and welcome message that fades out.
- Google Sign-In: Allow users to log in with Google Sign-In via Firebase Authentication.
- Fast Login: Implement a fast login setup with biometric or PIN authentication stored securely in local storage.
- Dashboard: Create a dashboard to view, add, and manage bills with offline support using localStorage or IndexedDB.
- Local Notifications: Provide options to set local notifications for bill payment reminders.
- Email Reminders: Support email reminders via Firebase, allowing users to receive bill payment reminders via email.

## Style Guidelines:

- Primary color: A muted blue (#5DADE2) evoking trust and reliability, aligning with the financial nature of the app. Avoids the cliche of green while maintaining a professional feel.
- Background color: Very light blue (#EBF5FB), subtly different from white, maintaining a clean, modern interface. 
- Accent color: Light violet (#A397D6), offering a contrast to the primary blue without being overly jarring, to be used sparingly for interactive elements and calls to action.
- Font: 'Inter' (sans-serif) for a modern, neutral, and highly readable interface. Suitable for both headlines and body text.
- Use minimalist, vector-based icons for bills, reminders, and user profile to maintain a clean and modern aesthetic.
- Employ a grid-based layout for the dashboard, ensuring consistent spacing and alignment for easy navigation and readability.
- Use subtle transitions and animations when adding, editing, or deleting bills to provide visual feedback without being distracting.
